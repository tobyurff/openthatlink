# Store Submission Assets

Assets formatted for browser extension store submissions.

## Chrome Web Store (`chrome/`)

| File | Size | Description |
|------|------|-------------|
| `icon-128.png` | 128x128 | Extension icon (96x96 icon with 16px transparent padding) |
| `promo-small.png` | 440x280 | Small promo tile |
| `promo-marquee.png` | 920x680 | Large/marquee promo tile |

### Chrome Requirements
- Icon must be 128x128 PNG with 96x96 actual content
- Small promo: 440x280 (optional but recommended)
- Marquee promo: 920x680 (optional)
- Screenshots: 1280x800 or 640x400 (add manually)

## Firefox Add-ons (`firefox/`)

| File | Size | Description |
|------|------|-------------|
| `icon-128.png` | 128x128 | Extension icon |

### Firefox Requirements
- Icon should be 128x128 PNG
- Screenshots recommended but not required for listing
